Deploying an eleventy site to github pages

To create an Eleventy website, you can follow these steps:

Step 1: Set up a Project

  1. Ensure that Node.js is installed on your computer. You can download and install it from the official Node.js website (https://nodejs.org).

  2. Open a command prompt or terminal and create a new directory for your project. Navigate to the directory using the cd command.

  3. Initialize a new Node.js project by running the following command:

    npm init -y

  4. Install Eleventy as a development dependency by running the following command:

    npm install --save-dev eleventy

  5. Create a new file named .eleventy.js in the root directory of your project. This file will be used for Eleventy's configuration.

Step 2: Configure Eleventy

In the .eleventy.js file, you can configure various settings for your Eleventy website. Here's an example configuration that you can modify to suit your needs:

module.exports = function (eleventyConfig) {
  eleventyConfig.addPassthroughCopy("assets"); // Copies the "assets" folder to the output directory

  return {
    dir: {
      input: "src", // Directory where your source files are located
      output: "dist", // Directory where the generated site will be outputted
    },
  };
};

Step 3: Create Your Site Content

  1. Create a new folder named src in the root directory of your project. This folder will contain your site's source files.

  2. Add Markdown or HTML files to the src folder to create your site's content. You can organize the files into subdirectories as needed.

Step 4: Build and Preview Your Site

  1. Open a command prompt or terminal and navigate to the root directory of your project.

  2. Run the following command to build your Eleventy site:

    npx eleventy

    This command will generate your site's HTML files and output them to the dist directory (or the directory specified in the configuration).

  3. After the build process completes, you can preview your site locally. You can use a static file server like http-server or live-server to serve the files. Alternatively, you can open the HTML files directly in your web browser.

Step 5: Customize and Enhance

You can customize your Eleventy website further by exploring the available features and plugins. Eleventy supports various templating languages, filters, and plugins to extend its functionality. Refer to the Eleventy documentation (https://www.11ty.dev/docs/) for more information and examples.

Remember to configure your project's Git repository, set up deployment options, and add any necessary styling or assets to make your Eleventy website complete.
